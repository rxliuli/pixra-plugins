# Grok Image Edit

A Pixra plugin that uses Grok AI's image generation API to edit images with natural language prompts.

## Features

- Edit images using natural language descriptions
- Powered by Grok's advanced AI image editing model
- Simple and intuitive prompt-based interface
- Supports background changes, object modifications, and style transformations

## Setup

Before using this plugin, you need to configure your Grok AI API credentials:

1. Get your API key from [x.ai](https://console.x.ai)
2. In Pixra, go to Settings â†’ Grok Image Edit
3. Enter your API key in the `Grok AI API Key` field

### Configuration Options

- **API Key**: Your Grok AI API key (required)

## Usage

1. Open an image in Pixra
2. Run the "Grok Edit Image" command from the Tools menu
3. Enter a description of how you want to edit the image
   - Example: "Change the background to a sunset beach"
   - Example: "Remove the person from the image"
   - Example: "Make it look vintage"
4. Wait for Grok AI to process the image
5. The edited result will replace your current image

## Examples

- "Change the background to a sunny beach scene"
- "Add snow to this landscape"
- "Make the image black and white"
- "Replace the sky with a starry night"
- "Remove the object in the center"

## Requirements

- Active Grok AI API subscription
- Internet connection for API calls

## Pricing

Image editing with Grok AI costs approximately **$0.02 per image**. Make sure you have sufficient credits in your x.ai account.
